# FitPartner

## Brief
상품 진열, 상품 구매, 게시글등록이 가능한 간단한 웹 쇼핑몰 입니다.
사용자는 아이디를 통해 판매자와 구매자로 나눠져있습니다.
#### 판매자
상품을 등록,수정,삭제 할 수 있으며 공지글을 작성할 수 있습니다.  
결제완료된 주문목록을 체크하고 상품 처리 관리를 할 수 있습니다.
#### 구매자
장바구니, 즉시구매를 이용하여 상품을 구매할 수 있습니다.
